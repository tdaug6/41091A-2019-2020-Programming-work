Here you will find all of the functions that we created custom this year. We have found all of these functions super helpful and we hope you do too.

 

PIDLoop(speed, distance, direction, time)

This function is a PID controller for driving forward and backwards. The parameters are the max speed (speed) measured in percentage, distance to travel (distance) measured in inches, the direction as a string statement of “fwd” or “rev” and time measured in sec.

NonPIDLoop(speed, distance, direction, time, stopMotor)

This function does not use a PID controller for driving forward and backwards. The parameters are the speed (speed) measured in percentage, distance to travel (distance) measured in inches, the direction as a string statement of “fwd” or “rev”, time the function has measured in sec, and the type of brake for the motors (stopMotor) as a string statement of “none” to have the motors keep running after the function and “yes” for a motor hold.
